# Исследование алгоритма K-Nearest Neighbors (KNN)

## О проекте
Данный проект является практическим исследованием метода k-ближайших соседей (KNN) в рамках домашнего задания курса по ML. 
Главная цель — пройти все этапы небольшого ML-проекта от разведочного анализа данных (EDA) до интерпретации результатов модели.

## Набор данных
Для классификации выбран датасет [**Wine**](https://www.kaggle.com/datasets/tawfikelmetwally/wine-dataset).
* **Описание**: Данные содержат результаты химического анализа вин, выращенных в одном регионе Италии, но тремя разными фермерами.
* **Задача**: Многоклассовая классификация (определение производителя вина на основе химических характеристик).
* **Особенности**: Содержит исключительно числовые признаки, что делает его идеальным для алгоритмов, основанных на вычислении дистанций.

## Структура репозитория
* `notebook.ipynb` — Jupyter Notebook с полным циклом исследования:
  * Первичный анализ данных (размер, типы, пропуски, баланс классов).
  * Подготовка данных (масштабирование, разбиение выборки).
  * Обучение базовой модели KNN.
  * Подбор гиперпараметров (число соседей, веса, метрики) с помощью кросс-валидации.
  * Интерпретация результатов и выводы.

## Запуск проекта
1. Склонируйте репозиторий.
2. Установите необходимые зависимости: `pip install pandas numpy scikit-learn matplotlib seaborn`.
3. Запустите Jupyter Notebook: `jupyter notebook` и откройте файл `.ipynb`.
