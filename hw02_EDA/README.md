# 📊 Exploratory Data Analysis (EDA):
## 📝 О проекте
Данный репозиторий содержит решение домашнего задания №2 по машинному обучению. 
Главная цель проекта — провести полный цикл разведочного анализа данных (EDA) на «сыром» датасете: от сбора и очистки до генерации новых признаков и проверки гипотез.

## Данные
В качестве исходных данных используется датасет **House prices**.
* **Источник:** https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data
* **Краткое описание:** Описание характеристик домов, выставленных на продажу.

## Выполненные задачи
В ходе работы над Jupyter Notebook были выполнены следующие шаги:
1. **Быстрый обзор данных:** Оценка размерности, типов данных и базовой статистики.
2. **Очистка данных:** Обработка пропусков (с обоснованием выбора стратегии заполнения/удаления) и поиск дубликатов.
3. **Расширенная статистика:** Расчет перцентилей, дисперсии, асимметрии (skewness) и эксцесса (kurtosis).
4. **Feature Engineering:**
   * One-Hot Encoding категориальных признаков.
   * Применение Feature Hashing для признаков с высокой кардинальностью.
   * Создание новых признаков на основе существующих.
5. **Визуализация:** Построение распределений (KDE), ящиков с усами (Boxplots) для поиска аномалий, диаграмм рассеяния и тепловой карты корреляций (Heatmap) с использованием `matplotlib`, `seaborn` и интерактивных графиков `plotly`.
6. **Формирование выводов:** Анализ полученных результатов и формулирование гипотез.
