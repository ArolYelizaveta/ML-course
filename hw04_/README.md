# Исследование линейной регрессии: Прогноз цен на Toyota Corolla

## Цель проекта
Построение модели машинного обучения для предсказания цены подержанных автомобилей Toyota Corolla. В проекте реализован полный цикл: от предобработки и очистки данных до сравнения регуляризованных линейных моделей (Ridge, Lasso).

## Описание набора данных
Датасет содержит информацию о 1436 автомобилях. 
- **Целевая переменная:** `Price` (цена в евро).
- **Признаки:** возраст автомобиля (`Age_08_04`), пробег (`KM`), мощность двигателя (`HP`), тип топлива, вес, объем двигателя и наличие различных опций (ABS, кондиционер и т.д.).

## Этапы работы
1. **EDA:** Анализ корреляций (выявлена сильная связь цены с возрастом и весом) и визуализация распределений.
2. **Feature Engineering:** Удаление неинформативных признаков (`Id`, `Model`), обработка категориального признака `Fuel_Type` через One-Hot Encoding, создание производного признака.
3. **Моделирование:** Сравнение стандартной линейной регрессии с моделями Ridge и Lasso. Подбор гиперпараметров выполнен с помощью кросс-валидации (`GridSearchCV`).
4. **Результаты:** Оценка качества по метрикам MAE, RMSE и R^2. Проверка моделей на переобучение.

## Выводы
Наилучший результат показала модель Ridge. Модель успешно объясняет около 90% дисперсии цены, что является высоким показателем для линейных методов на данных такого типа.
