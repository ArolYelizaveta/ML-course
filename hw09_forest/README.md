# Домашнее задание #9: Исследование Случайного Леса (Random Forest)

## Описание проекта
В данном проекте проводится исследование ансамблевой модели машинного обучения (Random Forest) для решения задачи регрессии — предсказания стоимости подержанных автомобилей Toyota Corolla. 

Работа является продолжением предыдущего исследования линейной регрессии. Главная цель — обучить деревянные модели, сравнить одиночное дерево решений (Decision Tree) со случайным лесом, а также сопоставить результаты леса с базовой линейной моделью.

## Датасет
Использован набор данных [Price of used Toyota Corolla cars](https://www.kaggle.com/datasets/vishakhdapat/price-of-used-toyota-corolla-cars). 
Целевая переменная (Target) — `Price` (цена автомобиля).

## Структура репозитория
* Jupyter Notebook с загрузкой данных, предобработкой, обучением моделей, расчетом метрик и развернутыми ответами на теоретические вопросы.
* `README.md` — описание проекта.

## Краткие выводы
* **Предобработка:** В отличие от линейной регрессии, случайный лес не требует масштабирования признаков (StandardScaler/MinMaxScaler), так как алгоритм основан на разбиении по порогам, а не на вычислении дистанций.
* **Скорость:** Одиночное дерево решений обучается значительно быстрее случайного леса, так как лес строит множество деревьев. Сократить разрыв в скорости можно за счет параллелизации (`n_jobs=-1` в sklearn).
* **Качество:** Случайный лес показал более высокую обобщающую способность по сравнению с одиночным деревом (снизилось переобучение) и лучше уловил нелинейные зависимости по сравнению с линейной регрессией.
